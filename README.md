# Simple Python Chatbot

A basic rule-based chatbot built using Python.

## Features
- Greeting responses
- Random jokes
- Time and date response
- Help command
- Exit with "bye"

## Sample Conversation
```
Bot : Welcome to Simple Chatbot!
Bot : Type 'bye' to exit.

You : hey
Bot : Hey!

You : how are you
Bot : I'm fine, thanks!

You : what is today's date
Bot : Today is Wednesday, 04-03-2026 and current time is 14:26

You : tell me jokes
Bot : What did the ocean say to the beach? Nothing, it just waved.

You : more jokes
Bot : Why did the student eat his homework? Because the teacher said it was a piece of cake.

You : another fun
Bot : Why do programmers prefer dark mode? Because light attracts bugs.

You : nice! another joke
Bot : Why did the math book look sad? Because it had too many problems.

You : lbw
Bot : I don't understand that. Try asking jokes.

You : jokes
Bot : Why did the student eat his homework? Because the teacher said it was a piece of cake.

You : bye
Bot : Goodbye!
```
